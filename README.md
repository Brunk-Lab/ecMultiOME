# ecMultiOME

This repository includes the codes used for analyzing NCIH2170 SNU16 COLO320DM COLO320HSR 10X Multiome (scRNA-seq + scATAC-seq) data and Bioskryb ResolveOME (scDNA-seq + scRNA-seq) data for NCIH2170, SNU16, COLO320DM, SKGT2, KATOIII, and COLO320HSR.
For Bioskryb ResolveOME analysis, there are two sequencing replicates (plates). The codes for each of the two replicates are included.
